##This is a File Handling Utility Java program.

It performs basic file operations:
1. Create a file
2. Read the file
3. Update the file

The program takes input from the user and stores it in a textfile.  
Then it shows the data and updates the file with a status message.

##How to run
1. Compile the program
2. Run the program
3. Enter your details

##Output
A file named student.txt will be created.
